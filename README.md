# scala-cli-gradle-spring-findbugs-mockito-testng-car-data

## Description
A POC for spring app using testng,
findbugs, mockito framework.

## Tech stack
- scala
- gradle
  - spring
  - testng
  - findbugs
  - mockito

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
